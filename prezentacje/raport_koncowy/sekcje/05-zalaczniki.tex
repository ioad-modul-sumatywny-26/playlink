\section{Załączniki}
\label{sec:zalaczniki}

Zgodnie z ustaleniami z prowadzącym niniejszy raport końcowy stanowi dokumentację
projektu. Poniższe materiały uzupełniające dołączono jako osobne pliki:

\begin{itemize}
  \item \textbf{Załącznik A — Dokumentacja techniczna} (plik
        \texttt{dokumentacja-techniczna.pdf}): szczegółowy, deweloperski opis
        architektury, pełnego API REST, protokołu WebSocket, modelu danych,
        konfiguracji, migracji oraz wdrożenia — scalone materiały z katalogu
        \texttt{docs/} repozytorium.
  \item \textbf{Załącznik B — Dokumentacja testów} (plik
        \texttt{dokumentacja-testow.pdf}): ta sama treść co rozdział~\ref{sec:testy}
        (\emph{Testowanie rozwiązania}), udostępniona dodatkowo jako osobny plik
        (sześć rodzajów testów oraz katalog dziesięciu przypadków testowych).
\end{itemize}
